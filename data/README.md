# Data Directory

This directory contains local source and generated working data. Curated,
versioned downloads for repository users live under `public-data/`.

Expected subdirectories:

- `raw/` - downloaded source files, not committed.
- `manual/` - small reviewed inputs committed with the repository.
- `processed/` - generated working files, not committed.
- `cache/` - temporary download cache, not committed.

Important raw inputs:

- `raw/election_results/` - official K17-K25 ballot rows fetched by
  `scripts/fetch_election_results.py`.
- `raw/cbs_historical_geography/` - official ballot crosswalks, 1995/2008/2011
  geometry, transition tables, and source manifest.
- `raw/ezorim_statistiim_2022.gdb/` - canonical 2022 statistical-area source.
- `raw/arcgis/elections2015_statistical_areas.geojson` and
  `raw/arcgis/elections2019_statistical_areas.geojson` - derivative layers used
  for labeled geometry supplements, display footprints, and aggregate
  reconstruction evidence; never as published vote totals.
- `raw/archive_knesset17_kalpies-list17-1.pdf` and
  `raw/archive_knesset17_kalpies-list17-2.pdf` - archived official K17 reports
  used to recover eligible-voter counts.

Important reviewed inputs:

- `manual/k17_eligible_voters.csv` - complete K17 ordinary-row denominators.
- `manual/locality_crosswalk.csv` - normalized historical/current locality
  identities and custom targets.
- `manual/ballot_geography_overrides.csv` - row-specific reviewed geographic
  classifications.
- `manual/arcgis_assignment_reconstruction_reviews.csv` - 44 approved Tier A
  locality-election reconstruction decisions. These authorize inferred
  ballot-to-area links while preserving official ballot and vote values.
- `manual/composite_localities.csv` and
  `manual/joined_locality_composites.csv` - election-specific locality unions.
- `manual/party_registry.csv` - election-specific party/list metadata.

Source preparation constraints:

- A fresh clone does not yet bootstrap every raw source automatically. The 2022
  FileGDB and archived K17 reports require preparation from
  `docs/DATA_SOURCES.md`.
- Statistical-mode pending rows are absent from the available historical
  ballot crosswalks. See `docs/PROJECT_STATUS.md`.

Do not use the old partial `statistical-areas-2022.geojson` export as a project
source. Raw downloads and working outputs remain ignored. The deliberate
exception is the reproducible release under `public-data/v1`, generated by
`scripts/build_public_data_release.py` and committed for direct download.
